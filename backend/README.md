# Backend API

Express.js backend for expense reimbursement system.

## Setup

1. Copy environment variables:
```bash
cp .env.example .env
```

2. Fill in your credentials in `.env`

3. Install dependencies:
```bash
npm install
```

4. Run development server:
```bash
npm run dev
```
