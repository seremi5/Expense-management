# Frontend Web App

React + Vite frontend for expense reimbursement system.

## Setup

1. Copy environment variables:
```bash
cp .env.example .env
```

2. Fill in your backend API URL in `.env`

3. Install dependencies:
```bash
npm install
```

4. Run development server:
```bash
npm run dev
```

5. Open http://localhost:5173
